## 1.자바 스크립트 코드 하이라이트 테스트입니다.
```javascript
// 구구단 출력
for(var i = 2; i < 9; i++) {
    for(var j = 1; j < 9; j++) {
        document.write(i + " * " + j + " = " + i * j);
    }
}
```

```
출력 결과
2 * 1 = 2
2 * 2 = 4
.
.
.
9 * 9 = 81
```
