## 1.자바 코드 하이라이트 테스트입니다.
```java
// 구구단 출력
for(int i = 2; i < 9; i++) {
    for(int j = 1; j < 9; j++) {
        System.out.println(i + " * " + j + " = " + i * j);
    }
}
```

```
출력 결과
2 * 1 = 2
2 * 2 = 4
.
.
.
9 * 9 = 81
```
